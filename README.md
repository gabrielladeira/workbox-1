# Work Box Application


## Create Application

###### STEP 0: 
- Install node.js


###### STEP 1:
- To Install Sails.js, open cmd & type the below command:

- ```npm install -g sails```


###### STEP 2:
- It wll create directory in:
- ```C:\git```


###### STEP 3:
- Create the app
- ```sails new workbox```
